# piecegluing
Implementation of enumeration of convex polygons constructed by gluing unit angle triangles.

## Required environment
* Java 17
* Maven
* Eclipse (optional)

## Build
```sh
mvn package
```

## Run
For 180/8 degree unit angle and 4-gon, Run:
```sh
cd target
java -jar piecegluing-0.0.2.jar 8 4 unique
```
